# ADR 0001: Platform Identity

## Status
Accepted

## Decision
Use `fintech-microservices-platform` as the repository, artifact, Docker image, and documentation identity. Use `com.rsingla.fintech` for Java packages.

## Consequences
The repository reads as a single polished fintech showcase instead of a folder of imported experiments. Legacy names such as `com.techcodepro`, `io.apicode`, and `SpringAutowire` are not carried forward.
